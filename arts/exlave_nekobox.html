Инструкция для новичков. Подходит для всех актуальных подписок.

  • Копируем ссылку на подписку
  • В приложении: плюсик -> импорт из буфера обмена
  • Три полоски -> КОНФИГУРАЦИЯ (на фото ошибка)
ВНИМАНИЕ

На этом моменте переключаемся на мобильный интернет с белым списком

  • Три точки -> тест подключения -> URL тест
  • После завершения теста: снова три точки -> сортировка -> по пингу (задержке)
  • Выбираем верхний сервер
  • Подключаемая, как на картинке, а также делаем тест

Если верхний сервер не работает - вы можете выбрать другой сервер с маленькой задержкой
